◆ motion_sensor_on()

void hardware_control::motion_sensor_on ( )
static

Turn the motion sensor on.

The RCWL-0516 motion sensor is powered by analog pin 0.

The motion sensor is powered up each time the feather goes to sleep. It's output goes high as it starts up potentially giving a false alarm. The interrupt guards and delayMicroseconds prevent this.
